## discription
译码器是对n个输入信号组成的编码进行译码，即输出2^n个信号，对于每个输入编码（n位，有2^n个编码）对应只有一个输出信号有效，其它2^n-1个输出无效。本实验要求设计一个3:8译码器，功能等同于集成电路器件74LS138.图1是74LS138的引脚图，表1为其功能表。
